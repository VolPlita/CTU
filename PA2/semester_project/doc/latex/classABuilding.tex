\doxysubsection{Detailed Description}
Base class for buildings. Building can be of five types. Residental house (Halupa), Lumber, Mine, Ambar or Fisher. Class saves information about buildings price, keeps reference of resource base that it will update and stores timer with time interval for it\textquotesingle{}s timer
